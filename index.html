Stage 0 Notes

World Wide Web

The web, invented in the early 90s is a large network of tons of computers that are all able to communicate with one another. Computers send requests to each other using HTTP(hypertext transfer protocol).

HTML

HTML (Hypertext Markup Language), is the language used to form the structure of most pages on the internet. HTML consists of text content and the markup which affects its layout.

Tags and Elements

HTML documents use tags like <b> and <p> to define the layout of the content. These tags in combination with their content makeup HTML elements. Most pages are made up of many elements. Most tags have content, but some tags like <img> do not contain content and are considered void.

Silly Computers

Computers, while powerful, are actually very simplistic and lack the intuition of humans. Computers only posess the ability to follow all orders literally and can therfore be very unforgiving of mistakes made in programmig.

Inline vs Block Elements

While inline elements, such as an image, do not affect the spacing, block elements place their content in an invisible box that can be modified.
